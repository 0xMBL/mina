# Archive Node 

# What is the archive node? 

# Prerequisites 
1. Have a working version of Coda installed. See the [docs](/docs/getting-started) 
2. Install Postgres [here](https://www.postgresql.org/download/). 

# How to run the archive node 
1. Create a new database in Postgres. 
2. Start the archive node and point it to the Postgres database. 
3. Start the coda daemon and point it to the archive node.
